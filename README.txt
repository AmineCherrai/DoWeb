What is DoWeb?
	DoWeb is a php framework for people who build dynamique websites using PHP. 
	DoWeb makes building websites easy and fast with simple and secure libraries and functions ready to use. 
	DoWeb is so easy to use, you don't need more than reading the User Guide to start building your website.

Release Information:
	DoWeb beta


Server Requirements:
	PHP version 5 or newer.
